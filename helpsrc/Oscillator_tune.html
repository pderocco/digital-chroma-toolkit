OSCILLATOR tune

These control the basic pitch of each {_oscillator} before modulation. Note that each oscillator is always modulated by the corresponding {_glide} output. The values are shown as the scale pitch heard when middle C is played; the default value is C5, which is middle C, 5 octaves above MIDI note zero.

Toolkit

These are contained in the {Oscillator_tune_cluster Oscillator tune clusters} within the {Oscillator_section}.

Chroma panel

These are contained in the PITCH section of the right panel, under the following parameter number:

{sw-26.png height=48}

These appear in the display as Osc Tune.

MIDI

These are accessed via the following {_NRPN NRPNs}:

A 1,36
B 2,36

Values

The values are displayed as scale pitches:
