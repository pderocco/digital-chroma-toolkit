SWEEP mode

These select one of five synchronization modes for the {_sweep} generators:

poly Each voice has its own sweep generator. The initial phase when a voice is allocated is random, and it runs continuously even if the voice is retriggered.
poly synced Each voice has its own sweep generator. The phase starts at 0\d at the start of each note.

mono All voices share the same sweep phase computed by the virtual instrument, but each has its own amplitude control. This phase runs continuously, independent of the playing of notes.

mono multiple Like the mono mode, except that the phase is reset to 0\d whenever a key is pressed.
mono single Like the mono mode, except that the phase is reset to 0\d whenever the number of keys held goes from zero to one.

Toolkit

These are contained in the {Sweep_rate_cluster Sweep rate clusters} within the {Sweep_section}.

Chroma panel
