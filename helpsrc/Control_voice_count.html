CONTROL voice count

This specifies the number of voices to be played for each note. If this is set to 1, the {Control_tune_spread} and {Control_pan_spread} parameters have no effect.

Toolkit

This is contained in the {Control_voice_stack_cluster voice stack cluster} within the {Control_section}.

Chroma panel

This is contained in the CONTROL section of the right panel, under the following parameter number:

{sw-set-split.png height=48} {sw-3.png height=48}

This appears in the display as Voice Count.

MIDI

This is accessed via {_NRPN} 0,25.

Values

MIDI display
1 1 (default)
2 2
3 3
4 4
