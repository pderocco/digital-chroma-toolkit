GLOBAL reverb room

This controls the reverb room used by the sound. On startup, and whenever it is changed, it injects a {MIDI_reverb_controls reverb room} control into the Main and Link {_instrument instruments}. MIDI playback instruments use the recorded value of this control, not this {_global_parameter}.

Note that this may be overridden by the {_program program's} {Control_reverb_room} {_parameter}.

Toolkit

This is contained in the {Global_parameters} section of the {Tools_screen}.

Chroma panel

This is accessed in {_program_select_mode PROGRAM SELECT mode} using the following global parameter number:

{sw-set-split.png height=48} {sw-1.png height=48}

This appears in the display as Reverb Room.

MIDI

This is accessed via {_NRPN} 15,1.
