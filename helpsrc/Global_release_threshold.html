GLOBAL release threshold

This adjusts the MIDI release velocity below which a key release is considered "slow" for the purpose of the {Envelope_release_mod_select} parameters. Low values make it easier to get a fast release, and higher values make it easier to get a slow release.

Toolkit

This is contained in the {Global_parameters} section of the {Tools_screen}

Chroma panel

This is accessed in {_program_select_mode PROGRAM SELECT mode} using the following global parameter number:

{sw-set-split.png height=48} {sw-5.png height=48}

This appears in the display as Release Threshold.

MIDI

This is accessed via {_NRPN} 15,5.
