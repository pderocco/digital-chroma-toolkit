CONTROL bass gain

This controls the gain of the bass section of the virtual {_instrument instrument's} parametric equalizer.

Toolkit

This is contained in the {Control_tone_cluster tone cluster} within the {Control_section}.

Chroma panel

This is manipulated by the BASS slider on the left panel. When any of the five short sliders is moved, the display shows their values for a couple seconds. Press PARAM SELECT if you want to see these settings without moving the sliders.

MIDI

This is accessed via {_NRPN} 0,16. Since it is a {_bipolar} parameter, the values are offset by 64.
