Program Info button: The second {_button} in the {_Navigation_bar}, which displays the number of the {_current_program}, and its name if it has one. The text in this button is normally light blue, but turns pink if the program is modified; it is yellow if a {_scratch_program} is selected.

{_long-press Long-pressing} the button displays a {_popup} that allows additional functions to be performed on the current program.
