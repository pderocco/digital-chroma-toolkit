FILTER mod select

These select a filter modulation source, using the same choices that are available for {_oscillator} and {_waveshaper} inputs. The "+" and "\-" lever options use only one side of the lever. The "thresh" options for the velocity and pressure yield no output until the input is at mid-scale, and then rise at twice the slope to the maximum value.

Toolkit

These are contained in the {Filter_mod_cluster Filter mod clusters} within the {Filter_section}.

Chroma panel

These are contained in the CUTOFF section of the right panel, under the following parameter numbers:

mod 1 mod 2 mod 3 mod 4 mod 5
