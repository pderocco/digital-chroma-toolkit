AMPLIFIER pan cluster

{amplifier-pan-cluster.png width=373}

The AMPLIFIER section has one of these for each {_channel}. It contains a {Amplifier_pan pan} {_parameter} that adjusts the left/right output position, a {Amplifier_pan_mod_select mod select} parameter to choose a modulation source, and a {Amplifier_pan_mod_depth mod depth} parameter to adjust the amount of modulation.

The pan parameter adjusts the balance between the left and right outputs. Its range is \-90\d (left) to +90\d (right), with a default value of 0\d.

The mod select parameter gives you the standard choices used by most modulation inputs. The default choice is {_sweep} B, for stereo tremolo effects.

The mod depth parameter adjusts how much of the range corresponds to one {_unit} of modulation input. It is off by default.
