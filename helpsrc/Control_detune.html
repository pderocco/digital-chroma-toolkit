CONTROL detune

This controls the amount by which {_oscillator} B is detuned relative to A. Oscillator A is unchanged.

Toolkit

This is contained in the {Control_detune_cluster detune cluster} within the {Control_section}.

Chroma panel

This is contained in the CONTROL section of the right panel, under the following parameter number:

{sw-4.png height=48}

This appears in the display as Detune.

MIDI

This is accessed via {_NRPN} 0,26. Since it is a {_bipolar} parameter, the values are offset by 64.

Values

The values are displayed in cents for the "full" {Control_detune_scale} setting, cents at middle C for the "half" setting, and in Hertz for the "fixed"; setting:
