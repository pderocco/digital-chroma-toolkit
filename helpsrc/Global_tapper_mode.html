GLOBAL tapper mode

This controls the Digital Chroma's panel {_tapper}. It has three settings: "off" to disable it; "switches" to enable it only for switch presses; and "on" to enable it for switch presses and for changes in the value of {_selector_parameter selector parameters}.

Toolkit

This is contained in the {Global_parameters} section of the {Tools_screen}.

Chroma panel

This is accessed in {_program_select_mode PROGRAM SELECT mode} using the following global parameter number:

{sw-set-split.png height=48} {sw-9.png height=48}

This appears in the display as Tapper Mode.

MIDI

This is accessed via {_NRPN} 15,9.

Values

MIDI display
0 off
1 switches
2 on
