CONTROL lever modes

This determines the selectivity mode of the two levers. In "selective" mode, when a key is released, the lever value freezes for that note, even if the note is held by the sustain {_footswitch}. It also freezes for notes {_latch latched} by the left footswitch. In "sampled" mode, it is sampled once at the start of the note.

Toolkit

This is contained in the {Control_selectivity_cluster selectivity cluster} within the {Control_section}. It is presented as two separate parameters, one for each lever.

Chroma panel

This is contained in the SWEEP section of the right panel, under the following parameter number:

{sw-set-split.png height=48} {sw-8.png height=48}

This appears in the display as Lever Modes.

MIDI

This is accessed via {_NRPN} 0,31.

Values

MIDI display
0 normal / normal
1 selective / normal
2 sampled / normal
3 normal / selective
4 selective / selective (default)
5 sampled / selective
6 normal / sampled
7 selective / sampled
8 sampled / sampled
