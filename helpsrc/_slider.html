slider: In the Toolkit, a wide rectangular {_control} with a colored bar in it that can be used to adjust a {_variable_parameter} by {_drag dragging} within it. The value can be reset to its {_default_value default} by {_long-press long-pressing} it, or in some cases set to some other significant value by {_tap tapping} it. In a few cases, a variable parameter may have a few values at the high end of the range for selecting certain special options, in which case a corresponding set of {_button buttons} will appear below the slider in the {_cluster} {_popup}.
