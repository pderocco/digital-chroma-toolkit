AMPLIFIER pan

These control the output pan position by adjusting the balance between the left and right outputs for each {_amplifier}. The values are displayed as angles, where \-90\d is full left and +90\d is full right.

Toolkit

These are contained in the {Amplifier_pan_cluster Amplifier pan clusters} within the {Amplifier_section}.

Chroma panel

These are contained in the VOLUME section of the right panel, under the following parameter number:

{sw-set-split.png height=48} {sw-46.png height=48}

They appear in the display as Pan.

MIDI

These are accessed via the following {_NRPN NRPNs}:

A 1,76
B 2,76

Since these are {_bipolar} parameters, the values are offset by 64.

Values

The internal numeric values are displayed as angles:

MIDI display
0 \-90\d
\b\n\b\n\b \b\n\b\n\b
64 off (default)
\b\n\b\n\b \b\n\b\n\b
127 +90\d
