COMMON edit mode

This controls whether the {_parameter_control} is connected to an A {_parameter}, a B parameter, or both in parallel. It has no effect if the parameter currently selected on the Chroma's control panel isn't one of the channelized parameters.

Toolkit

This is contained in the {Common_edit_cluster edit cluster} within the {Common_section}.

Chroma panel

This is set with the following switches:

{sw-edit-a.png height=48} A
{sw-edit-b.png height=48} B
{sw-edit-a.png height=48} + {sw-edit-a.png height=48} A\&B

The two switches must be pressed concurrently for A\&B.

MIDI

This is accessed via {_NRPN} 0,9.

Values

MIDI mode
1 Edit A
2 Edit B
3 Edit A\&B (default)
