GLOBAL midi record out

This selects the port and channel pair to send Main and Link output on, for recording in an external sequencer.

Toolkit

This is contained in the {Global_parameters} section of the {Tools_screen}. It has separate buttons for the port and for the channels. When you tap a channel button, it selects it and the next higher channel.

Chroma panel

This is accessed in {_program_select_mode PROGRAM SELECT mode} using the following global parameter number:

{sw-set-split.png height=48} {sw-8.png height=48}

This appears in the display as MIDI Record Output.

MIDI

This is accessed via {_NRPN} 15,8.
