GLIDE mod select

These select one of 16 modulation sources for the time of the {_glide} generators. The "+" and "\-" lever options use only one side of the lever.

Toolkit

These are contained in the {Glide_cluster Glide clusters} within the {Glide_section}.

Chroma panel

These are contained in the GLIDE section of the right panel, under the following parameter number:

{sw-set-split.png height=48} {sw-6.png height=48}

These appear in the display as Glide Mod Select.

MIDI

These are accessed via the following {_NRPN NRPNs}:

A 1,2
B 2,2
