MIDI text sysex

This is ASCII text with no control characters. The {MIDI_Edit_Response Edit Response} {_NRPN} is preceded by a {_Sysex} containing an identification string. The {MIDI_Undo_Redo_Status Undo/Redo Status} NRPN is preceded by a Sysex containing two concatenated description strings, to indicate the next undo and redo in line. A Text Sysex is never used for input to the Digital Chroma.
