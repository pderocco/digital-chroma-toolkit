GLOBAL midi link in

This selects the port and channel to accept Link input on. This is the input that is used when the {_link_program link} mode is set to "link external" or "link internal". The former uses the external input to play the Link sound, while the internal keyboard plays the Main sound. The latter does the reverse.

Toolkit

This is contained in the {Global_parameters} section of the {Tools_screen}. It has separate buttons for the port and for the channels.

Chroma panel

This is accessed in {_program_select_mode PROGRAM SELECT mode} using the following global parameter number:

{sw-set-split.png height=48} {sw-7.png height=48}

This appears in the display as MIDI Link Input.

This is accessed via {_NRPN} 15,7.
