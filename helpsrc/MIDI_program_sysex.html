MIDI program sysex

This contains a complete Digital Chroma {_program}. It begins with the same ten bytes as a {MIDI_metadata_sysex}:

  • Year minus 2000 (0 to 99)
  • Month (1 to 12)
  • Day (1 to 31)
  • Hour (0 to 23)
  • Minute (0 to 59)
  • Second (0 to 59)
  • 10ms (0 to 99)
  • 100\us (0 to 99)
  • The program architecture, currently always 0
  • The program version, currently always 0

It is then followed by a complete set of parameters:

  • 36 bytes of Common and Control parameters
  • 79 bytes of A channel parameters
  • 79 bytes of B channel parameters

This may be followed by one or more lines of text {MIDI_metadata_sysex metadata}.
