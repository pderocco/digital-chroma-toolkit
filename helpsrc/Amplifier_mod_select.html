AMPLIFIER mod select

These select a primary modulation source for modulating the volume of each {_amplifier}. These sources are the {_envelope envelopes}, but there is also a "fixed" setting that provides a constant 1 {_unit} value, which is useful when the {_patch} uses Amplifier B to adjust the mix between the A and B {_oscillator oscillators}.

Toolkit

These are contained in the {Amplifier_mod_cluster Amplifier mod clusters} within the {Amplifier_section}.

Chroma panel

These are contained in the VOLUME section of the right panel, under the following parameter numbers:

mod 1 mod 2
{sw-46.png height=48} {sw-48.png height=48}

They appear in the display as Amp Mod1/2 Select.

MIDI

These are accessed via the following {_NRPN NRPNs}:

mod 1 mod 2
A 1,70 1,72
B 2,70 2,72

Values

MIDI display
0 envelope 1A (mod 1 default)
1 envelope 2A
2 envelope 1B (mod 2 default)
3 envelope 2B
4 fixed
