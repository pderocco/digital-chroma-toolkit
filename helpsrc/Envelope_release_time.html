ENVELOPE release time

These control the basic release time of each {_envelope} generator, before modulation, from instantaneous to over 20 seconds.

Toolkit

These are contained in the {Envelope_release_cluster Envelope release clusters} within the two {Envelope_sections}.

Chroma panel

These are contained in the ENVELOPE 1 and ENVELOPE 2 sections of the right panel, under the following parameter numbers:

Env 1 Env 2
{sw-18.png height=48} {sw-25.png height=48}

These appear in the display as Env1/2 Release.

MIDI

These are accessed via the following {_NRPN NRPNs}:

Env 1 Env 2
A 1,21 1,34
B 2,21 2,34

Values

MIDI display
0 instant (default)
1 2ms
