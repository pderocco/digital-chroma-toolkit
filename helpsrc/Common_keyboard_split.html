COMMON keyboard split

This defines the bottom key of the upper portion of the keyboard, when the {Common_link_mode} is set to one of the split keyboard modes.

Toolkit

This is contained in the {Common_link_cluster link cluster} within the {Common_section}.

Chroma panel

This is set by pressing:

{sw-set-split.png height=48}

and then pressing the desired key. You can also press

{sw-set-split.png height=48} {sw-set-split.png height=48}

to set it to the same point it was last set to by this mechanism.

MIDI

This is accessed via {_NRPN} 0,4.
