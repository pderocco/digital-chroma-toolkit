CONTROL tune spread

This controls how much stacked voices are detuned among each other. The detuning is a half-scale detuning where the beat frequency doubles for every two-octave pitch rise. This has no effect unless the {Control_voice_count} parameter is greater than 1.

The value is the maximum number of cents deviation at middle C. For instance, if set to 6, the detunings for each voice count setting would be:

1: 0\c
2: \-3\c \m \m \m \m \m +3\c
3: \-3\c \m \m 0\c \m \m +3\c
4: \-3\c \m \-1\c \m +1\c \m +3\c

The top two settings connect one of the {_pedal pedals} to the spread.

Toolkit

This is contained in the {Control_voice_stack_cluster voice stack cluster} within the {Control_section}. Buttons beneath the slider select the pedal settings.

Chroma panel

This is contained in the PITCH section of the right panel, under the following parameter number:

{sw-set-split.png height=48} {sw-27.png height=48}

This appears in the display as Tune Spread.

MIDI

This is accessed via {_NRPN} 0,17.
