GLOBAL rollover time

This sets the delay when playing in certain keyboard algorithms, to allow a complete chord to be acquired before assigning the notes to {_voice voices}. The higher this is set, the more delay you'll feel, but the sloppier you can be about playing chord notes simultaneously.

Toolkit

This is contained in the {Global_parameters} section of the {Tools_screen}

Chroma panel

This is accessed in {_program_select_mode PROGRAM SELECT mode} using the following global parameter number:

{sw-set-split.png height=48} {sw-4.png height=48}

This appears in the display as Rollover Time.

MIDI

This is accessed via {_NRPN} 15,4.
