sequence program: A parameter that refers to the number of a program that should be selected when the SEQUENCE PROGRAM footswitch is tapped. This allows a sequence of programs to be set up in advance, and stepped through easily in performance. In the Digital Chroma, it is set by holding the footswitch and selecting the program while it is held. In the Toolkit, it is set by long-pressing the target program button in the Programs screen, and tapping the “sequence” button.

The default value of the parameter is previous, which reloads whatever program was selected (including any unsaved edits) before the current one. This program can always be stepped to by long-pressing the footswitch, which provides a way to recover if the footswitch is tapped accidentally.
