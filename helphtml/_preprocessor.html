preprocessor: The software module in the Digital Chroma that splits the notes from the keyboard into main and link streams, does main and link octave transpositions, performs keyboard algorithms like arpeggiation, and handles the footswitch and ribbon mode parameters.

When recording a sequence in an external DAW via the USB device port MIDI, it is the output of the preprocessor that is recorded. This means that the USB device port MIDI input doesn’t perform those preprocessor functions, or support those parameters, since those functions were already performed during the recording.
