panel mode: In the Digital Chroma, the basic mode that the control panel is in, which determines what the numbered switches do. The mode normally used while playing is PROG SELECT mode, in which the numbers select programs. While editing, the panel is in PARAM SELECT mode, in which the numbers select parameters and connect them to the PARAMETER CONTROL. The COPY FROM A and COPY FROM B modes allow parameter values to be copied from another program, or between the A and B parameters.

The panel mode has no bearing on the Toolkit.
