slider: In the Toolkit, a wide rectangular control with a colored bar in it that can be used to adjust a variable parameter by dragging within it. The value can be reset to its default by long-pressing it, or in some cases set to some other significant value by tapping it. In a few cases, a variable parameter may have a few values at the high end of the range for selecting certain special options, in which case a corresponding set of buttons will appear below the slider in the cluster popup.
