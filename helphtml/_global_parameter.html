global parameter: A numeric value that can be adjusted by the user, which controls some aspect of the overall operation of the Digital Chroma, and is not part of a program. Examples are MIDI channel selection, and the lever polarities. From the Digital Chroma’s panel, they can be selected by pressing SET SPLIT and a number in the top row while in program select mode, and then varied with the parameter control. In the Toolkit, they can be adjusted directly in the Tools screen.
