USB host port: Either of the two USB type A connectors on the Digital Chroma’s rear panel. They can be used for flash drives, for access using the various LOAD and SAVE buttons on the control panel, or for MIDI input from external keyboards, pedalboards, or controllers. When used for MIDI, the upper one is called “HostA” and the bottom “HostB”. The global MIDI Main Input parameter selects one channel on either port for merging in with the internal keyboard and performance controls, and the MIDI Link Input parameter selects another channel that can be used for independent input when the link mode parameter is set either to “external” or “internal”.
