parameter: A numeric value that is part of a program, that controls some single aspect of a sound. Every parameter has a name, a range of internal numeric values which are used over MIDI, a default value in a scratch program, and a more user-friendly representation of each possible numeric value. For instance, an oscillator pitch is shown using note symbols like “G#4”, while a modulation selector value is shown as the name of the modulation source, like “envelope 2A”.

Parameters are categorized as selector parameters, which have distinct settings, and variable parameters, which represent a continuous range of values with a certain resolution. On the Digital Chroma, parameters are also categorized as “regular”, if they were included in the original Chroma, or “extended”, if they are new.
