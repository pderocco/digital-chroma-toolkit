SET SPLIT: The switch on the Digital Chroma’s control panel that is used to set the keyboard split parameter, but also used as a generalized prefix for accessing secondary functions of the remaining switches.

The keyboard split point is set by pressing SET SPLIT, and then pressing the key that is to be the bottom of the upper part of the keyboard. SET SPLIT is also used before a numbered switch in Parameter Select mode to select an extended parameter, or in Program Select mode to select a global parameter or calibration.
